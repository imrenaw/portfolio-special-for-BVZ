Route Optimizer

Full-stack route optimization app demonstrating a custom Dijkstra implementation, FastAPI backend, and Leaflet.js map visualization.

Highlights

  • Custom Dijkstra algorithm for shortest paths
  • FastAPI backend serving node & edge data
  • Interactive Leaflet map for route preview

Demo: (if backend running) open the map interface to draw routes and visualize the computed path.
