Powerful CSS Features

Modern CSS provides powerful tools for creating beautiful, responsive, and interactive designs without relying on JavaScript. Here are some essential modern CSS techniques.

2. CSS Flexbox & Grid

Flexbox

Perfect for one-dimensional layouts. Easily align and distribute items.

CSS Grid

Ideal for complex two-dimensional layouts with rows and columns.

Responsive

Both adapt automatically to different screen sizes.

/* Flexbox - Perfect for navigation */ .navbar { display: flex; justify-content: space-between; align-items: center; gap: 20px; } /* CSS Grid - Perfect for page layout */ .page-layout { display: grid; grid-template-columns: 1fr 2fr 1fr; gap: 20px; /* Results in: sidebar | main | sidebar layout */ }

3. CSS Custom Properties (Variables)

Define and reuse values throughout your CSS

CSS variables make it easier to maintain consistent spacing, colors, and typography across your entire site.

/* Define custom properties */ :root { --primary-color: #FF8C00; --secondary-color: #FFD700; --spacing-unit: 8px; --border-radius: 8px; } /* Use them anywhere */ button { background: var(--primary-color); padding: calc(var(--spacing-unit) * 2); border-radius: var(--border-radius); } /* Easy theme switching */ @media (prefers-color-scheme: dark) { :root { --primary-color: #ffaa33; --secondary-color: #ffe066; } }

4. Transforms & Transitions

This element has a subtle shadow (hover over cards to see smooth transitions)
/* Smooth transitions */ .element { transition: all 0.3s ease; transform: translateY(0); } .element:hover { transform: translateY(-10px); box-shadow: 0 20px 40px rgba(0, 0, 0, 0.1); } /* Complex transforms */ .box { transform: rotate(45deg) scale(1.1) skew(10deg); }

5. Advanced Selectors

/* Child combinator */ .parent > .child { } /* Sibling combinator */ .element + .sibling { } /* Attribute selector */ input[type="email"] { } /* Pseudo-classes */ a:hover, a:active, a:visited { } li:nth-child(odd) { } li:last-of-type { } /* Pseudo-elements */ p::before { content: ""; } p::first-letter { font-size: 2em; }

Benefits of Modern CSS:

  • Less JavaScript needed for styling
  • Better performance
  • More maintainable code
  • Easier to create responsive designs
  • Beautiful animations and interactions
  • Better browser support
