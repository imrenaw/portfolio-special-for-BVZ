NumPy Arrays & Operations

Array Operations Demo

>>> import numpy as np
# Create arrays
>>> arr1 = np.array([1, 2, 3, 4, 5])
>>> arr2 = np.array([10, 20, 30, 40, 50])
# Element-wise operations
>>> arr1 + arr2
array([11, 22, 33, 44, 55])
# Array slicing
>>> arr1[1:4]
array([2, 3, 4])

NumPy allows fast array operations without explicit loops

Matrix Operations

>>> # 2D Array (Matrix)
>>> matrix = np.array([[1, 2], [3, 4]])
matrix:
[[1 2]
[3 4]]
>>> # Matrix multiplication
>>> np.dot(matrix, matrix)
array([[ 7, 10],
[15, 22]])
>>> # Transpose
>>> matrix.T
array([[1, 3],
[2, 4]])

Matrix operations are essential for linear algebra and machine learning

Broadcasting

Broadcasting allows NumPy to work with arrays of different shapes:

>>> a = np.array([[1, 2], [3, 4]]) # shape (2, 2)
>>> b = np.array([10, 20]) # shape (2,)
# Broadcasting adds b to each row of a
>>> a + b
array([[11, 22],
[13, 24]])

This feature eliminates the need for explicit loops - huge performance boost!
