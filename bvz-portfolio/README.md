# BVZ Portfolio

## Overview
This project is a modern portfolio website for "BVZ" that showcases skills, projects, and contact information. The website features an impressive loading animation of the typography "BVZ" and dynamic transformations as the user scrolls down the page. It is designed with a premium, slightly futuristic style, utilizing a dark color palette and smooth scrolling effects.

## Features
- Animated "BVZ" typography on loading
- Sections for About, Skills, Projects, and Contacts
- Dynamic transformations and smooth scrolling effects
- Responsive design for various screen sizes
- Clean and extensible code structure

## Project Structure
```
bvz-portfolio
├── src
│   ├── index.html          # Main HTML structure of the portfolio
│   ├── styles
│   │   ├── main.css       # Primary styles for layout and typography
│   │   ├── animations.css  # CSS animations for typography and elements
│   │   └── responsive.css  # Media queries for responsive design
│   ├── scripts
│   │   ├── main.js        # Main JavaScript logic and event listeners
│   │   ├── animations.js   # Handles typography animations and transitions
│   │   └── scroll.js      # Implements smooth scrolling and parallax effects
│   └── assets
│       └── fonts          # Custom fonts for enhanced typography
├── package.json           # npm configuration file
└── README.md              # Project documentation
```

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd bvz-portfolio
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage
- Open `src/index.html` in a web browser to view the portfolio.
- The website features smooth scrolling and dynamic animations as you navigate through the sections.
